# Video Trellis

Draw the scenes of a video as an animated grid of small multiples.

![Design for Dreaming](design-for-dreaming-1950.gif)
MPO Productions. (1950). Design for Dreaming [Film]. Internet Archive. https://archive.org/details/Designfo1956

## CLI

```
video-trellis --input path/to/file --loop-clips --resolution 2048x1080
```
